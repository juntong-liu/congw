# congw

## 页面介绍

这是一个动物主题的静态网页示例，当前主页面为 `animal.html`。  
页面以“小猫档案”为内容核心，采用卡通可爱风设计，包含：

- 渐变背景与装饰元素，提升整体视觉氛围
- 角色头图区（猫咪形象）与标题副标题展示
- 信息卡片化布局（名称、性格、习惯、食物）
- 轻量动效（入场动画、头像浮动、按钮反馈）
- 移动端自适应（在小屏下自动单列展示）

## 使用方式

1. 克隆仓库后直接打开 `animal.html`
2. 或使用任意本地静态服务器预览页面

## 适用场景

- HTML/CSS 页面练习
- 前端作品展示
- 动物主题网页模板参考
